프로젝트 소개

기간 : 2023.08.28~10.12

목적 : 자바프로젝트를 웹으로 개발. 반려동물 종합 케어 서비스

개발 인원 : 5명

담당 파트 : 반려동물 정보 관리

프로젝트 설계

반려동물 정보 관리

1. 내 반려동물 조회

youmka/petc

세션의 회원 아이디 정보를 가져와 데이터베이스에서 조회해 반려동물 정보 반환

yomuka/petc

1. 반려동물 수정 및 삭제

2. 반려동물 등록

2. 반려동물 정보 등록

yomuka/petc/register1

반려동물 대분류 선택

yomuka/petc/register2

1. 반려동물 사진 등록
2. 이름 입력창
3. 나이, 나이 단위 입력창
4. 몸무게 입력창
5. 중성화 여부 입력
6. 주요질병 여부 입력창
7. 품종 입력창
8. 유효성 검사 후, 반려동물 등록 후 메인 페이지로 이동

3. 반려동물 정보 수정 / 삭제

yomuka/petc/update?petNum=?

1. 데이터베이스에 저장되어 있는 기존 정보
2. 2번 버튼 action시 해당 반려동물 삭제
3. 데이터 작성 후 3번 버튼 action시 반려동물 데이터 수정

사용기술 및 도구

Databse

- H2 Database

Web

- HTML5, CSS3, BootStrap5, Maven, Apache-Tomcat

Tool
